# 小学英文单词复习工具 - 产品文档

## 1. 项目概述

**名称**: 英文单词记忆卡 (English Word Cards)

**目标用户**: 小学生 (6-12岁)

**主要功能**: 帮助学生通过卡片式学习、练习和测试来掌握英文单词

**核心价值**: 让单词学习变得有趣、互动、高效

**使用场景**: 纯移动端应用 - 可随时随地在手机上打开学习 (5-10分钟碎片化学习)

---

## 2. 产品目标

- 提供趣味化的单词学习体验
- 增强学生的单词记忆能力
- 追踪学习进度
- 支持多种学习模式
- 适配移动端和桌面端

---

## 3. Phase 1 核心功能 (听音写单词模式)

### 3.1 单词库管理

**支持的单元 (3个)**:
- 🏠 家庭成员 (Family) - 15个单词
- 🍎 食物 (Food) - 15个单词
- 🦁 动物 (Animals) - 15个单词

**每个单词包含**:
- 英文单词
- 中文翻译
- 发音音频文件 (MP3)

**数据存储**: JSON格式，内置在HTML中或单独JS文件

### 3.2 听音写单词模式 (Dictation Mode)

**功能流程**:

#### 第1步：选择参数
```
选择单元 → 家庭成员 / 食物 / 动物
选择单词数 → 随机选择 5/10/15 个单词
```

#### 第2步：听音和写单词
```
界面显示:
┌─────────────────┐
│ 单词 1/15       │
│                 │
│  [播放] 🔊      │  <- 可重复点击播放音频
│                 │
│  输入框: ______ │  <- 输入英文单词
│                 │
│  下一个 →       │  <- 完成一个词后才能点
└─────────────────┘

规则:
- 隐藏英文，只能看中文翻译
- 可以无限次点击播放按钮听发音
- 每个单词必须输入至少1个字符才能下一个
- 支持输入全部正确 / 部分输入 / 完全错误
```

#### 第3步：自己检查
```
当所有单词都输入完毕后，显示答案列表:

标题: 听写答案检查

┌──────────────────────────┐
│ 1. 单词: apple          │
│    你的输入: aple       │
│    [改错]               │  <- 可修改答案
├──────────────────────────┤
│ 2. 单词: banana        │
│    你的输入: banana    │
│    ✓ 正确!              │  <- 正确显示对勾
├──────────────────────────┤
│ 3. 单词: cat           │
│    你的输入: (空)      │
│    [需要输入]           │  <- 未输入
└──────────────────────────┘

底部按钮:
[改错]  [完成]
```

#### 第4步：最终结果
```
完成后显示:

┌──────────────────────┐
│   听写完成!           │
│                      │
│ 正确: 15/15 ✓       │
│ 准确率: 100%        │
│                      │
│ 详细列表:           │
│ ✓ apple - apple    │
│ ✗ banana - bananna │
│ ✓ cat - cat        │
│                      │
│ [再来一次]  [首页]  │
└──────────────────────┘
```

### 3.3 关键交互设计

**播放音频**:
- 点击绿色播放按钮 🔊 播放发音
- 可无限次重复
- 音频使用Web Audio API或<audio>标签

**输入框**:
- 英文字母输入框 (只允许a-z)
- 自动转小写
- 实时显示输入内容

**检查答案**:
- 进入检查界面后，用户可以修改任何答案
- 对比规则: 完全匹配为正确，否则为错误
- 可点击单词进行修改

**正确率计算**:
- 正确 = 与标准答案完全匹配
- 公式: (正确数 / 总数) × 100%

## 9. Phase 1 用户激励 (简化版)

这版本先不加复杂激励，重点是功能本身的趣味性。

### 9.1 基础反馈机制

**为什么这样就够了**:
- 听音频本身就很有趣（像谜题）
- 输入答案有成就感
- 看到自己的正确率会有满足感

**即时反馈**:
- 播放音频时，按钮有动画效果
- 输入时，实时显示字数
- 完成时，显示正确率百分比

### 未来版本(Phase 2+)才考虑的激励系统

#### 1. 连续打卡 (Streak System)
```
🔥 连续学习 5 天
显示当前连续天数的火焰图标
目标: 让学生养成每日学习习惯
```
- 每天只需学习1个单元就能维持打卡
- 中断后重新开始计数
- 达成里程碑显示特殊成就

#### 2. 等级和经验 (Leveling System)
```
👤 小学员  → 单词小卫士 → 英文高手 → 单词大师
Lv.1      Lv.5          Lv.10       Lv.20
10000 XP   25000 XP      50000 XP    100000 XP
```
- 每学会一个单词 +10 XP
- 每通过一次测试 +50 XP
- 达到新等级时有升级动画和音效
- 等级越高，个人页面上显示的勋章越多

#### 3. 成就徽章系统 (Achievements)
```
🥇 新手上路     - 学完第一个单元
🌟 知识宝宝     - 掌握50个单词
🚀 冲刺高手     - 连续学习7天
💯 完美分数     - 一次测试得100分
🔥 学习狂人     - 一天学超过3个单元
📚 全能学者     - 完成所有单元
```
- 完成条件清晰可见
- 获得时有弹出庆祝动画
- 可分享给家长和朋友

#### 4. 随机奖励盒子 (Surprise Rewards)
```
完成一个单元 → 打开随机宝箱
- 50% 概率: 金币 +5
- 30% 概率: 星星 +3
- 15% 概率: 特殊动画表情
- 5%  概率: 隐藏成就解锁
```
- 增加期待感和随机性
- 即使学习困难也有惊喜

#### 5. 排行榜和社交
```
本周学习王者:
1. 🥇 小明 - 150 XP
2. 🥈 小红 - 130 XP
3. 🥉 小刚 - 120 XP
你: 第 25 名 (↑3)
```
- 与同学比较 (有健康的竞争)
- 显示周排名变化
- 鼓励而非贬低

## 12. Phase 1 为什么这个功能有趣？

### 听音写单词的吸引力

1. **挑战性** - 像一个谜题需要破解
   - 不是单纯的选择题，需要自己拼写
   - 有难度但不会太难 (毕竟看得到中文提示)
   - 孩子会积极想"我能猜对吗？"

2. **即时反馈** - 立刻知道对错
   - 不用等待家长或老师批改
   - 自己检查答案，自己知道掌握度

3. **成就感** - 看到正确率提升
   - 从第一次50%到第二次80%，会很开心
   - 可以自己多练几次来提高成绩

4. **声音的吸引力** - 声音比文字更生动
   - 听发音本身就很有趣
   - 可以多听几遍来加深印象

### 对比其他模式

- ❌ 纯选择题：太简单，没有成就感
- ❌ 纯卡片翻转：被动接收，缺乏参与感
- ✅ 听音写单词：积极参与、有挑战、有反馈
