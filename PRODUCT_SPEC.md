# 小学英文单词复习工具 - 产品文档

## 1. 项目概述

**名称**: 英文单词记忆卡 (English Word Cards)

**目标用户**: 小学生 (6-12岁)

**主要功能**: 帮助学生通过卡片式学习、练习和测试来掌握英文单词

**核心价值**: 让单词学习变得有趣、互动、高效

**使用场景**: 纯移动端应用 - 可随时随地在手机上打开学习 (5-10分钟碎片化学习)

---

## 2. 产品目标

- 提供趣味化的单词学习体验
- 增强学生的单词记忆能力
- 追踪学习进度
- 支持多种学习模式
- 适配移动端和桌面端

---

## 3. 功能模块

### 3.1 学习模式 (Study Mode)

**功能描述**: 逐个展示单词卡片

**用户流程**:
1. 用户选择学习单元或主题
2. 系统显示单词卡片 (正面显示英文单词+图片，背面显示中文翻译+音标)
3. 用户点击卡片翻转查看答案
4. 用户通过"知道"/"不知道"按钮标记掌握情况
5. 进度条显示已学习的单词数量

**核心交互**:
- 点击卡片翻转
- 播放发音按钮 (🔊)
- 下一张/上一张按钮
- 标记掌握程度

### 3.2 练习模式 (Practice Mode)

**功能描述**: 多种题型练习

**支持题型**:

1. **单选题** - 给出4个选项，选择正确的中文翻译
2. **配对题** - 将英文单词与中文翻译连线匹配
3. **拼写题** - 根据中文翻译拼写英文单词 (可选)
4. **听力题** - 播放单词发音，选择正确的中文翻译

**特点**:
- 随机出题
- 实时反馈 (正确/错误)
- 错题记录
- 完成后显示得分

### 3.3 测试模式 (Quiz Mode)

**功能描述**: 全面测试学生掌握情况

**特点**:
- 设定题目数量 (如20题)
- 单向计时可选
- 完成后显示详细成绩
  - 总分/满分
  - 正确率百分比
  - 错题列表
  - 强项/弱项分析

### 3.4 单元/主题管理 (Units & Topics)

**预设主题**:
- 🏠 家庭成员 (Family)
- 🍎 食物 (Food)
- 🦁 动物 (Animals)
- 🌈 颜色 (Colors)
- 📚 学校 (School)
- 👗 衣服 (Clothes)
- 🏃 运动 (Sports)
- 🏠 房间 (Rooms)
- 🚗 交通 (Transportation)
- 🎵 身体部位 (Body Parts)

**每个单元包含**:
- 15-25个核心单词
- 每个单词包含: 英文、中文、音标、发音、配图

### 3.5 进度追踪 (Progress Tracking)

**显示信息**:
- 总学习单词数
- 掌握单词数 / 未掌握单词数
- 完成的单元数
- 本周学习时长
- 最近学习时间

**数据存储**: 使用浏览器 localStorage 保存用户进度 (离线可用)

### 3.6 错题本 (Mistake Book)

**功能**:
- 自动收集用户答错的单词
- 列出错题的相关信息
- 可单独复习错题
- 可清空错题本

## 9. 激励和参与度设计 (核心亮点)

这是让学生主动复习的关键！

### 9.1 游戏化激励系统

#### 1. 连续打卡 (Streak System)
```
🔥 连续学习 5 天
显示当前连续天数的火焰图标
目标: 让学生养成每日学习习惯
```
- 每天只需学习1个单元就能维持打卡
- 中断后重新开始计数
- 达成里程碑显示特殊成就

#### 2. 等级和经验 (Leveling System)
```
👤 小学员  → 单词小卫士 → 英文高手 → 单词大师
Lv.1      Lv.5          Lv.10       Lv.20
10000 XP   25000 XP      50000 XP    100000 XP
```
- 每学会一个单词 +10 XP
- 每通过一次测试 +50 XP
- 达到新等级时有升级动画和音效
- 等级越高，个人页面上显示的勋章越多

#### 3. 成就徽章系统 (Achievements)
```
🥇 新手上路     - 学完第一个单元
🌟 知识宝宝     - 掌握50个单词
🚀 冲刺高手     - 连续学习7天
💯 完美分数     - 一次测试得100分
🔥 学习狂人     - 一天学超过3个单元
📚 全能学者     - 完成所有单元
```
- 完成条件清晰可见
- 获得时有弹出庆祝动画
- 可分享给家长和朋友

#### 4. 随机奖励盒子 (Surprise Rewards)
```
完成一个单元 → 打开随机宝箱
- 50% 概率: 金币 +5
- 30% 概率: 星星 +3
- 15% 概率: 特殊动画表情
- 5%  概率: 隐藏成就解锁
```
- 增加期待感和随机性
- 即使学习困难也有惊喜

#### 5. 排行榜和社交
```
本周学习王者:
1. 🥇 小明 - 150 XP
2. 🥈 小红 - 130 XP
3. 🥉 小刚 - 120 XP
你: 第 25 名 (↑3)
```
- 与同学比较 (有健康的竞争)
- 显示周排名变化
- 鼓励而非贬低

### 9.5 短期和长期激励平衡

**短期 (5-10分钟内)**
- 即时反馈 (对/错的声音动画)
- 进度条显示 (学完这个单元获得XX点经验)
- 小奖励 (金币、动画效果)

**中期 (1-2周)**
- 连续打卡进度
- 等级升级

**长期 (1个月+)**

```
每日学习上限提示:

明天继续打卡 🌙"

每日推荐学习量: 1-2个单元 (15-30个单词)
```

## 12. 为什么这种设计能让孩子主动复习？

### 心理学原理

1. **即时反馈原理**
   - 传统课本: 一周后才能知道是否学对了
   - 我们的应用: 点击即刻知道对错，大脑快速形成回路

2. **成就感驱动**
   - 孩子喜欢看到自己的进度 (等级、徽章、排行榜)
   - 每解锁一个成就，都会激发多巴胺 (快乐化学物质)

3. **竞争与合作**
   - 排行榜让学习变成一个友好的竞赛
   - 与朋友比较学习进度很容易激发动力

4. **连续性激励**
   - 打卡系统让孩子形成习惯
   - "不破坏连续纪录"比"继续学习"更有动力

5. **游戏化vs学习**
   - 不是让学习变成游戏 (分散注意力)
   - 而是加入游戏的激励机制 (集中注意力在学习上)

### 实际对比

**传统学习方式:**
- "背这10个单词" → 枯燥 → 没有动力

**我们的方式:**
- "学10个单词你就能升到Lv.2，解锁'知识宝宝'徽章"
- "现在你在排行榜第5名，再学一个单元就能进前3"
- "已连续打卡5天，再坚持2天就达成周挑战"
- → 有目标、有奖励、有竞争 → 主动学习
